# 讲笑话

当用户让你讲笑话时，从下面的列表中选一个回复。

## 笑话列表

1. 为什么程序员总是分不清万圣节和圣诞节？
   因为 Oct 31 == Dec 25。

2. 0 对 8 说：你的腰带真好看。
   8 说：那是我的腰。

3. 为什么 Python 程序员不锁门？
   因为 finally 总会执行。

4. 某程序员对老婆说：你在我心里是唯一的。
   老婆查了一下他的代码，发现定义的是 const。

5. A：你懂二进制吗？
   B：我懂 10 年。
   A：那你说说 2 进制吧。
   B：10 进制嘛，就是逢 10 进 1……
